# E--Library-Management-System
A simple console-based Library Management System written in C.  
This program allows users to store, search, and display book records using a menu-driven interface.  
It is useful for learning **structures, arrays, strings, loops, and switch-case in C programming**.


**objective**

* A C-based E-Library program to store and manage book records.
* Allows adding, displaying, and searching books by author.
* Stores details like book name, author, pages, and price using structures.
* Menu-driven program with options to view total number of books.
* Supports up to 100 books and runs in a loop until user exits.

**Features**
| Option | Description |
|--------|-------------|
| 1 | Add new book details |
| 2 | Display all stored books |
| 3 | List books written by a specific author |
| 4 | Show total number of books in the library |
| 5 | Exit from the program |
